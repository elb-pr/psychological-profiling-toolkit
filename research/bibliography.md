# Research Foundation — Pigeon Superstition Superposition

## Architecture
Every skill file is supported by exactly **5 sources**, each occupying a defined slot:

| Slot | Purpose |
|------|---------|
| 1 | **Founding paper** — original that established the framework |
| 2 | **Instrument / operationalisation** — how it is measured in validated use |
| 3 | **Validation / replication** — independent evidence the framework holds |
| 4 | **Indirect application precedent** — evidence this framework can be inferred without direct access |
| 5 | **Documentary proxy methodology** — linguistic, behavioural, or content-analytic markers specifically |

*S9 and S10 use a modified slot 5: cross-section synthesis methodology instead of documentary proxy, as those sections are derived not directly inferred.*
